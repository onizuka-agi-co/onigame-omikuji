# onigame-omikuji

Tiny static omikuji app for GitHub Pages.

## Features

- one-click fortune draw
- five fortune tiers with short messages
- lucky item and lucky color suggestions
- recent draw history stored in `localStorage`

## Files

- `index.html` - app markup
- `styles.css` - visual design
- `app.js` - draw logic and history

## GitHub Pages

This repo is designed to work as-is on GitHub Pages.
Deploy the repository root as a static site.
